Apache Isis™ is a framework to let you rapidly develop domain-driven apps in Java. This app contains entities that hold all possible data types and layouts — in other words a "kitchen sink" — to act as a regression suite for the framework.

To access the app:

  • wicket/

    provides accesses to a generic UI for end-users and administrators.

  • restful/

    provides access to a RESTful API conformant with the Restful Objects spec.

The default user/password is sven/pass.
